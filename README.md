##### 進め方
1. **1日1時間**の読書をする  
   - 集中力を保つために、25分読書 → 5分休憩 → 25分読書 → 最後の5分でメモ・まとめ、というフローを基本にします。  
2. **内容や感想を簡単に記録する**  
   - 日ごとにフォルダやマークダウンファイルを作成し、学習内容・気づいたことをメモします。  
   - 例：`Day1/summary.md` / `Day2/summary.md`  
3. **疑問点は随時メモして調べる**  
   - 本を読んでいて出てきた専門用語や気になるキーワードは軽く調べて追記します。
